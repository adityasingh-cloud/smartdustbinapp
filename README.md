# SmartBin — AI Waste Management 🗑✨

> **By Team Leavron** | Brutalist Editorial × IoT Dashboard

A stunning mobile app UI for **SmartBin** — an AI-powered smart waste management system. Built with React + Vite + Three.js.

## 🎨 Design Language

Brutalist editorial magazine aesthetic meets IoT dashboard:
- **Mustard Yellow** `#E8C547` · **Off-White** `#F5F0E8` · **Deep Charcoal** `#1A1A18` · **Rust Orange** `#C4622D`
- Typography: Bebas Neue (display) · DM Mono (techy mono) · Space Grotesk (body)

## 📱 Screens

| Screen | Features |
|--------|----------|
| **Dashboard** | 3D rotating SmartBin (Three.js), fill level bars, EcoCoins counter, activity feed, GPS map |
| **Camera** | Radar sweep animation, AI waste detection simulation, category pills |
| **Profile** | 3D parallax avatar tilt, stats, reward cards, bar chart, settings toggles |
| **Alerts** | Color-coded alerts, liquid fill animation, GPS map snippet, buzzer indicator |
| **Map** | Interactive bin map, SVG road grid, pulsing bin pins, nearby bins list |
| **Rewards** | EcoCoins balance hero, brand voucher cards, community leaderboard |

## ✨ Animations

- **3D SmartBin** — Three.js WebGL rotating model with glowing yellow/green/blue compartments
- **Liquid fill bars** — CSS animation filling like water pouring
- **EcoCoins counter** — count-up spring animation on load
- **Radar sweep** — sonar pulse rings + rotating sweep line on Camera screen
- **Parallax tilt** — 3D CSS tilt on Profile avatar (mouse/touch)
- **Card spring physics** — staggered slide-up on screen load
- **Floating particles** — hexagonal particles drifting in the dark background
- **Toggle switches** — spring bounce animation

## 🚀 Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view the app.

## 🛠 Tech Stack

- **React 18** + **Vite**
- **Three.js** — 3D bin model with WebGL (CSS fallback)
- **Google Fonts** — Bebas Neue, DM Mono, Space Grotesk
- **Pure CSS animations** — no external animation libraries

## 🌍 Built for

SmartBin is an AI-powered smart waste segregation system that uses computer vision to classify waste (Dry / Wet / Metal), rewards eco-friendly behavior with **EcoCoins**, and provides real-time bin monitoring for municipalities.
